# Gradient descent

Gradient descent updates parameters using \(x_{t+1}=x_t-\eta \nabla f(x_t)\). The minus sign matters: it points us downhill on the loss surface.

A small learning rate improves stability but can be slow. A large learning rate can overshoot the minimum or diverge.
